# 校园闲置品交易系统 java913

## 项目概述

校园闲置品交易系统致力于为校园内的学生提供一个便捷、高效的闲置物品交易平台。系统主要包含后台管理功能与前台用户功能，通过精细化的模块划分，实现信息的发布、管理及交易全流程。

## 功能模块

### 后台功能

#### 管理员功能

- 管理员管理：查看、新增、修改和删除管理员账号
- 基础数据管理
  - 闲置品类型管理：维护闲置品分类
  - 公告类型管理：维护公告分类
  - 学生会员等级管理：维护会员等级
  - 卖家信用类型管理：维护卖家信用等级
- 求购社区管理：管理求购帖子
- 闲置品管理
  - 闲置品信息管理：审核、查看、新增、修改、删除闲置品
  - 闲置品评价管理：查看、修改、删除用户评价
  - 闲置品收藏管理：查看、删除用户收藏
  - 闲置品订单管理：查看、处理订单
- 公告信息管理：发布和管理公告
- 卖家管理：审核、管理卖家账号
- 学生管理：管理普通学生用户
- 轮播图管理：管理首页轮播图

#### 卖家同学功能

- 求购社区管理：查看、新增、删除求购帖子
- 闲置品管理
  - 查看自己发布的闲置品
  - 管理自己闲置品的评价（查看、修改）
  - 查看自己闲置品的收藏情况
  - 查看和处理自己闲置品的订单
- 公告信息管理：查看系统公告

### 前台（买家同学）

- 个人中心功能
  - 闲置品收藏：管理用户收藏的闲置品
  - 闲置品订单：查看和管理用户的闲置品交易订单
- 闲置求购论坛：发布和浏览求购信息
- 闲置品：浏览和搜索闲置品
- 系统公告：查看系统发布的相关公告

## 系统角色

- 管理员：负责整个系统的维护与管理工作
- 卖家：发布闲置品、管理自己的交易活动等
- 买家：浏览、购买闲置品，参与求购论坛交流
